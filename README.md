# Sea Ice Extent

a simple script to create one file of all the mean months of extent sea ice in square km.

## Link

The Sea-ice extent data, from 1973 to 2025 is from [https://www.meereisportal.de](https://www.meereisportal.de) (funding: REKLIM-2013-04)

[seaiceportal.de](https://data.seaiceportal.de/relaunch/extent.php?region=s&minMonth=1&maxMonth=12&submit1=New+search&timeline=extent&active-tab1=measurement&ice-type=extent&lang=en&active-tab2=extent)
